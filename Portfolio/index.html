Aqui, você terá a oportunidade de explorar e conhecer de perto os projetos nos quais tenho dedicado meu tempo e habilidades. Cada projeto é mais do que um simples conjunto de códigos e pixels; são resultados tangíveis da minha paixão pela tecnologia e dedicação ao desenvolvimento web.

  • Pokédex- Enciclopédia Pokémon

    Como ele funciona? Ele utiliza a PokeAPI para buscar informações de um Pokémon e exibi-las em uma página HTML. O código começa selecionando os elementos HTML relevantes, como o nome do Pokémon, o número do Pokémon e a imagem do Pokémon.
    Em seguida, ele define um formulário para capturar a entrada do usuário, um botão "Anterior" e um botão "Próximo" para navegar entre os Pokémon.
    A variável "searchPokemon" é inicializada como 1. A função assíncrona "fetchPokemon" é definida para buscar os dados do Pokémon usando a PokeAPI. Ele recebe o número do Pokémon como parâmetro e faz uma solicitação GET para a API usando o número do Pokémon.
